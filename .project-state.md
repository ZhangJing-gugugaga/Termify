# 项目状态：Termify

## 一句话定义
万物皆可终端——把 GIF/PNG/JPG 转成终端可播放动画（.py / .html），并规划嵌入式输出与公开在线 Demo；开源（许可证待定：作者已要求删除 MIT License 段，暂无 MIT 执照，待重新讨论并选定）、零注册、零云端。

## PRD 摘要
- **核心功能（v1.0 已完成）**：① 5 种渲染风格（ascii / blocks / braille / geometric / binary）② 输出 .py 终端播放 + .html 浏览器播放 ③ Web 界面 + CLI 双形态 ④ 全屏自适应缩放 + 同目录 music.mp3 自动播音频 ⑤ REST API（upload/preview/generate/download）
- **本版本不做（v1 范围外，已规划）**：视频直传（v2）、嵌入式输出（v2）、公开在线 Demo（v2 部署项）
- **商业模式**：开源（许可证待定）、零注册、社区驱动 + 技术影响力资产；不重投入硬商业化、不限流
- **技术栈**：Python 3.10+ / Flask / Pillow 后端，原生 HTML/CSS/JS 前端，pytest（实测 123 tests）
